# 教学统计应用 - 本地安装指南

## 简介
教学统计应用是一个纯前端Web应用，用于记录和统计师生上课情况。本应用无需服务器支持，所有数据存储在浏览器本地，可在任何现代浏览器中运行。

## 安装步骤

### 方法一：直接下载并运行（推荐）

1. **下载应用文件**
   - 从提供的链接下载 `teaching_stats_app.zip` 压缩包
   - 将压缩包解压到您选择的文件夹中

2. **运行应用**
   - 打开解压后的 `teaching_stats_app` 文件夹
   - 找到并双击 `index.html` 文件
   - 应用将在您的默认浏览器中自动打开

### 方法二：使用Git克隆（适合开发者）

如果您熟悉Git，可以通过以下命令克隆项目：

```bash
# 克隆项目到本地
git clone https://github.com/yourusername/teaching_stats_app.git

# 进入项目目录
cd teaching_stats_app

# 直接在浏览器中打开index.html文件
# 或使用本地服务器运行
```

### 方法三：使用本地服务器（可选）

对于更专业的使用，您可以使用本地服务器运行应用：

1. **使用Python简单HTTP服务器**
   ```bash
   # 进入项目目录
   cd /path/to/teaching_stats_app
   
   # 启动Python服务器（Python 3.x）
   python -m http.server 8000
   
   # 在浏览器中访问
   # http://localhost:8000
   ```

2. **使用Node.js的http-server**
   ```bash
   # 安装http-server（如果尚未安装）
   npm install -g http-server
   
   # 进入项目目录
   cd /path/to/teaching_stats_app
   
   # 启动服务器
   http-server -p 8000
   
   # 在浏览器中访问
   # http://localhost:8000
   ```

## 数据存储说明

- **本地存储**：应用使用浏览器的 `localStorage` 存储数据，所有数据保存在您的本地设备上
- **数据持久性**：除非您手动清除浏览器数据，否则数据将一直保留
- **数据备份**：建议定期使用应用内的"导出数据"功能备份您的数据
- **数据恢复**：如需恢复数据，可使用"导入数据"功能上传之前导出的JSON文件

## 浏览器兼容性

应用支持所有现代浏览器：
- Google Chrome (推荐)
- Mozilla Firefox
- Microsoft Edge
- Safari

## 常见问题

### Q: 应用无法正常运行怎么办？
A: 请确保您使用的是最新版本的浏览器，并尝试清除浏览器缓存。

### Q: 如何备份我的数据？
A: 在应用中点击左侧导航栏底部的"导出数据"按钮，下载JSON文件并妥善保存。

### Q: 如何恢复我的数据？
A: 在应用中点击左侧导航栏底部的"导入数据"按钮，选择之前导出的JSON文件。

### Q: 数据会同步到其他设备吗？
A: 不会，数据仅存储在您当前使用的设备和浏览器中。如需在多设备间同步，请手动导出和导入数据。

## 技术支持

如果您在使用过程中遇到任何问题，请检查以下几点：
1. 确保您的浏览器是最新版本
2. 清除浏览器缓存和Cookie
3. 尝试在不同的浏览器中运行应用
